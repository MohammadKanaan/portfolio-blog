### How to fix issues with api routes not appearing in `php artisan routes:list`:
```PHP
return Application::configure(basePath: dirname(__DIR__))
    ->withRouting(
        web: __DIR__.'/../routes/web.php',
        api: __DIR__.'/../routes/api.php',
        commands: __DIR__.'/../routes/console.php',
        health: '/up',
    )
```
and to circumvent in the future just use the artisan command instead of manually adding the api.php
```BASH
php artisan install:api
```
